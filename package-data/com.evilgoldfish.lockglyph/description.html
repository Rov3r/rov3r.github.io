UNOFFICIAL BETA WITH iOS 10 SUPPORT


Adds the Apple Pay completed payment animation to the lockscreen, and animates when unlocking the device with Touch ID.


About this fork:

This code has been modified by @AppleBetasDev and @nullriver to add support for iOS 10. Full credit for everything else goes to @evilgoldfish01. In addition, this is an early build (hence the "beta"), so it will likely contain a lot of bugs. Feel free to report any to us, as long as they aren't in the list below.


Known issues:

  • Notifications do not scroll up far enough to accomodate LockGlyph
  • Notifications can cover LockGlyph and it will not hide according to iOS 10 lock screen UX
  • Some localizations are missing
  • LockGlyph does not show up if some tweaks are installed
  • Album art covers LockGlyph
  • Themes appear too zoomed in
  • Default LockGlyph position does not accomodate for Press home text if enabled
  • LockGlyph will reset its state if unlocked (while on lock screen) and Siri is activated
  • Landscape X position setting doesn't yet work
